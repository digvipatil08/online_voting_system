# 🗳️ Online Voting System

A simple **PHP + MySQL based Online Voting System** that allows users to register as voters, vote for candidates, and view live results. Includes an admin panel to manage candidates and voters.

## 🚀 Features

- 🧑‍🤝‍🧑 Voter Registration with image upload  
- 🗳️ Candidate listing with profile and party symbol  
- 🔐 Secure voting (each voter can vote once)  
- 📊 Result dashboard showing live vote counts  
- 🧑‍💻 Admin login to manage voters and candidates  
- 🎨 Modern UI using **Bootstrap 5** and smooth animations

## 🛠️ Technologies Used

- Frontend: HTML, CSS, Bootstrap 5, JS  
- Backend: PHP (MySQLi)  
- Database: MySQL  
- Server: XAMPP / Apache
